# 🐣 Тамагочі-бот для Телеграму (Cloudflare Workers)

Абсурдно-фановий віртуальний улюбленець, який живе у Телеграмі. Доглядай за ним, балакай із ним — а він капризує, жартує і **сам нагадує про себе**, коли засумує.

**Один спільний бот для всіх.** Ти розгортаєш його **один раз**, а далі будь-хто просто відкриває бота в Телеграмі й одразу отримує власного тамагочі. Жодного встановлення для користувачів.

**Вартість ≈ $0:** Cloudflare Workers + база D1 + cron — усе в межах безкоштовного тарифу. Між повідомленнями нічого не працює й не витрачає ресурсів.

## 🚀 Розгортання (робиться ОДИН раз, тобою)

Тобі знадобляться: акаунт Cloudflare (безкоштовний), токен бота і безкоштовний AI-ключ. ~10 хвилин.

### Крок 1. Створи бота в Телеграмі
Напиши **@BotFather** → `/newbot` → скопіюй **токен** (`123456:ABC...`).

### Крок 2. Візьми безкоштовний AI-ключ
Зареєструйся на **https://openrouter.ai** → **Keys → Create Key** → скопіюй (`sk-or-...`).
*(Картку прив'язувати не треба — стартуємо на безкоштовних моделях.)*

### Крок 3. Розгорни на Cloudflare

[![Deploy to Cloudflare](https://deploy.workers.cloudflare.com/button)](https://deploy.workers.cloudflare.com/?url=https://github.com/oleond93/tamagotchi-bot)

Натисни кнопку → увійди в Cloudflare → воно склонує репозиторій тобі й створить базу D1 автоматично.

**Або через термінал** (якщо зручніше):
```bash
npm install
npx wrangler d1 create tamagotchi      # скопіюй database_id у wrangler.toml
npx wrangler deploy
```

### Крок 4. Додай 2 секрети
У дашборді: **Workers → tamagotchi-bot → Settings → Variables and Secrets** додай:
- `TELEGRAM_TOKEN` = токен із кроку 1
- `OPENROUTER_API_KEY` = ключ із кроку 2

*(через CLI: `npx wrangler secret put TELEGRAM_TOKEN` і `... OPENROUTER_API_KEY`)*

### Крок 5. Увімкни бота
Відкрий у браузері адресу свого воркера з `/init` на кінці:
```
https://tamagotchi-bot.<твій-субдомен>.workers.dev/init
```
Побачиш «✅ Вебхук налаштовано» — все. Напиши боту `/start` у Телеграмі 🎉

## 🎮 Що вміє бот

- **Стан спадає з часом** — і набір потреб залежить від стадії: тепло 🔥, ситість 🍗, чистота 🛁, енергія ⚡, настрій 🎉, психіка 🤪, спілкування 🫂, сенс буття 🌌. Занепад швидкий (≈1.5–2 год до нуля), але **вночі сповільнюється** (22:00–08:00), щоб пет пережив сон.
- **Догляд кнопками** під карткою (`/status`) — кнопки самі змінюються під стадію. Є й слеш-команди (`/feed`, `/play`, `/sleep`, `/clean`…).
- **Еволюція за віком:** 🥚 Яйце → 🫠 Слизька Грудка → 👾 Дивне Створіння → 😈 Хаотичний Монстр → 🐉 Стародавнє Зло → 🌌 Космічна Сутність. Кожна стадія відкриває нові потреби, дії та події.
- **Вдача** фіксується при вилупленні — залежить від того, як добре ти грів яйце (😇 Янгол … 😈 Дике).
- **Характер** (7 типів: ґремлін, філософ, сонечко, драма, сонько, ботанік, бунтар) дістається **випадково** при вилупленні, далі **формується від догляду** (може стати сумішшю двох) і застигає на стадії 😈. Він впливає не лише на тон, а й на **гру — як RPG-клас**: пасиви (різна швидкість занепаду показників), синергія дій (улюблені дії дають більше, нелюбі — менше) і **фірмова здібність** на окремій кнопці (кулдаун). Дивитись: `/personality`.
- **Вільний чат** — відповідає «в характері» з урахуванням стану, вдачі, стадії й пори доби.
- **Памʼять 🧠** — у чаті памʼятає контекст останніх реплік і **сам запамʼятовує факти про тебе** (імʼя, уподобання, події, близькі, плани), щоб згадувати їх потім — і навіть у нагадуваннях. Подивитись, що запамʼятав: `/memory` або кнопка «🧠 Памʼять». *(Натхнення — підхід Hermes Agent; усе лишається в межах безкоштовного тарифу — без зайвих викликів AI.)*
- 🎲 **Події**, 🏆 **досягнення** (23 шт.), денний стрік 🔥.
- **Сам пише першим** — і не одноманітно: «режисер» проактивності обирає тип виходу на звʼязок за контекстом — ранкове привітання 🌅, порятунок серії 🔥, згадка факту з памʼяті 🧠, відлуння емоції 😤, спонтанна думка 💭 чи потреба 🍗. Одне повідомлення на годину; уночі не турбує.

## 🔑 Налаштування AI-провайдера

За замовчуванням — безкоштовний OpenRouter. Хочеш свій провайдер (будь-який OpenAI-сумісний)? Додай секрети замість OpenRouter:

| Змінна | Призначення |
|---|---|
| `LLM_BASE_URL` | напр. `https://api.твій-провайдер.com/v1` |
| `LLM_API_KEY` | твій ключ |
| `LLM_MODEL` | назва моделі |

`LLM_*` мають пріоритет над OpenRouter.

| `src/egg.js` | контент і дрібні взаємодії стадії яйця |
| `src/guide.js` | вбудований лор/гайд (гортається сторінками) |
